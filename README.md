## ⚠️ Wicked Demise Graveyard Tracker

**This addon is designed for the Wicked Demise guild on Skullcrusher EU.** It tracks graveyards and death locations across delve content using player-generated gravestones.

***

## Introduction

Hello! This addon helps Wicked Demise members track and submit in-game death locations.

It allows players to interact with gravestones created after deaths in delves and submit them to the guild’s website for tracking and statistics.

***

## How to Use

1.  Log into World of Warcraft
2.  Enter any delve level in the game
3.  Find gravestones created by players after death
4.  Right-click a gravestone
5.  Copy the popup link
6.  Paste it into your browser to submit the death record to the guild website 🎉

***

## Website Features

Once submitted, death data is displayed on the Wicked Demise guild website with features such as:

*   Latest Fallen
*   This Week’s Carnage
*   Top Nemesis
*   Deadliest Delve

And much more community-driven tracking content.

***

## Additional Features

*   Monthly death tracking and statistics
*   Weekly Discord summaries of in-game deaths
*   Armory integration for player context
*   Item tooltip enhancements for better insight into player gear
*   Historical death logs and progression tracking

***

## Commands

There are **no in-game commands** for this addon.

All interaction happens through the UI and gravestone interactions.
